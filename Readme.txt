Requisitos para execucao - 

A API foi desenvolvida na versão 2.6.7 springBoot FrameWork e na versão 11 de java, necessário um jdk 11 para execucao;

IDE - Visual Studio code:
necessário a extenção do spring boot para inicializar a API (Spring Boot Extension Pack);

IDE - Eclipse (Spring Tool Suite 4);

Banco de Dados - Mysql
caso o seu main user do Mysql seja diferente de root root, login e senha respectivamente, é necessario alterar no seguinte diretorio da API
src>main>resources>application.properties
na linha (2) temos o usuario;
na linha (3) temos a senha;
